# Vaishanvi Nidhane Portfolio

**Technologies:** HTML5 | CSS3 | JavaScript

## Overview
This is a **responsive personal portfolio website** built using HTML5, CSS3, and JavaScript.  
It highlights my **projects, skills, and contact information** with smooth layouts, animations, and interactive features.  
The portfolio follows **modern UI/UX principles** to provide an engaging and intuitive user experience.

## Features
- Fully **responsive design** for desktop, tablet, and mobile devices  
- **Smooth scrolling navigation** and functional contact form  
- **Interactive animations** for better user engagement  
- **Optimized performance** and accessibility for fast load times  
- **Downloadable resume** for professional presentation
